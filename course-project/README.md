
## Description

Social network powered by NestJS & PostgreSQL

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Config

```.env

APP_PORT

DB_HOST
DB_NAME
DB_USER
DB_PASSWORD
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
